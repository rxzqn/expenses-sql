# Expenses SQL Tracker

Personal finance tracker built with SQLite.

## Features
- Add expenses with category
- Add income with source
- View current balance
- View expenses grouped by category
- Data saved automatically

## How to Run
1. Install Python 3
2. Download `expenses_sql.py`
3. Run in terminal: `python expenses_sql.py`

## Built With
- Python 3
- SQLite3 for data storage
